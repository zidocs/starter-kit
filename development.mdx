---

Follow these steps to install and run ‘Zidocs’ on your operating system:

Step 1: Install Zidocs:

```bash
npm i -g zidocs
```
Step 2: Navigate to the docs directory (where the zidocs.json file is located) and execute the following command:

```bash
zidocs dev
```
