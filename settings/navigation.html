Tabs

Tabs help distinguish between different topics or sections of your documentation. They show up above the main sidebar. Example: DocumentationDocumentation Writing ContentWriting Content

Configure tabs with the tabstabs field of the zidocs.jsonzidocs.json file. The urlurl field of the tab object should map to a folder of content added to your sidebar, or an external link.

"tabs": [
    {
        "name": "API References",
        "url": "api-references"
    },
    {
        "name": "Content",
        "url": "content"
    },
    {
        "name": "Blog",
        "url": "https://your-website.com/blog"
    }
]

To configure the default DocumentationDocumentation primary tab, add the primaryTabprimaryTab field to your zidocs.jsonzidocs.json file with your desired name. Any files in your navigation not in a folder reserved by another tab will show up in the primary tab.

"primaryTab": {
    "name": "Home"

Sidebar

Our sidebar navigation syntax is recursive which means you can make nested navigation groups. You don't need to include .mdx.mdx in page names.

Once you add a page to your docs directory, you'll need to add the path to zidocs.jsonzidocs.json to add it to the sidebar. Pages do not show up automatically.

Regular Navigation
"navigation": [
    {
        "group": "Getting Started",
        "pages": ["quickstart"]
    }
]
Regular Navigation
"navigation": [
    {
        "group": "Getting Started",
        "pages": ["quickstart"]
    }
]

Folders

Simply put your MDX files in folders and update the paths in zidocs.jsonzidocs.json.

For example, to have a page at https://yoursite.com/your-folder/your-pagehttps://yoursite.com/your-folder/your-page you would make a folder called your-folderyour-folder containing an MDX file called your-page.mdxyour-page.mdx.

You cannot use apiapi for the name of a folder unless you nest it inside another folder. Zidocs uses Next.js which reserves the top-level apiapi folder for internal server calls. We recommend using the folder name api-referenceapi-reference instead.

Navigation With Folder
"navigation": [
    {
        "group": "Group Name",
        "pages": ["your-folder/your-page"]
    }
